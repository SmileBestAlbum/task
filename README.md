# Установка

Для работы этого проекта требуется установить следующие библиотеки:
- `selenium`
- `pytest`

Вы можете установить их с помощью команды pip:

### _Установка Selenium_
- `pip install selenium`

Selenium требует файл chromedriver.exe, и ему нужно знать, где его найти. Здесь есть несколько возможных подходов, но давайте добавим его в переменную PATH Windows, как описано в документации Selenium.

Возьмите скачанные файлы, скопируйте их в C:\WebDriver\bin\. Затем запустите это в командной строке Windows:

- `setx PATH "%PATH%;C:\WebDriver\bin"`

### _Установка Pytest_
- `pip install pytest` - для установки pytest


# Запуск тестов

1. ### Откройте терминал в корневой папке репозитория.

2. ### Запустите тесты с помощью команды:
- `pytest test_avito_count.py`
- `pytest -v test_avito_count.py` - для получения подробностей по каждому из тестов

3. ### **Ожидаемые результаты**

- После выполнения тестов проверьте папку output для сохраненных скриншотов.

- При успешном выполнении тестов скриншоты счетчиков будут сохранены в папке output.

- Если тесты не прошли из-за отсутствия элементов на странице, сообщения об ошибках будут сохранены в файле BUGS.md.


##### Дополнительная информация
В случае возникновения проблем или вопросов, пожалуйста, обратитесь к [документации Selenium WebDriver](https://selenium-python.readthedocs.io/index.html).

При необходимости можно настроить другие параметры запуска браузера, такие как размер окна и режим headless, в файле test_avito_count.py.
